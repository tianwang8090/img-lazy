# lazyImg
An image lazyload tool for web.

## install & use

### install with npm or `srcipt` tag
`npm i lazyImg -S` or `<script src=""></script>`

### use
```html
<script src=""></script>

<!-- set the real img source value in 'data-src' -->
<img data-src="x.jpg">

```
```javascript
// with import
import lazyImg from "lazyImg";

lazyImg();
```
